# Amira App

Complete Expo React Native project for the Amira Social App.

## Batch 1 & 2 Implementation
- **Batch 1 (Onboarding & Auth):** Splash, Welcome, Phone Login, OTP, Profile Setup (Name, Birthday, Gender, Photos, Interests, Location).
- **Batch 2 (Main Features):** Discover (Swiping), Moments (Social Feed with Blur), Messages (List & Detail with Video Call), Profile, Wallet (Diamonds), VIP Store, Settings.

## Tech Stack
- Expo / React Native
- React Navigation (Native Stack & Bottom Tabs)
- Lucide React Native (Icons)
- Mock Services (Firebase, Agora, Paystack)

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Run `npx expo start` to launch the app.
